Internship at UPV

Role

Software Developer & Researcher

Overview

Development of face-tracking software using Unity and Python to support accessibility in healthcare. The project enables children with mobility problems to interact with Tella, an educational application developed at HVL Medielab.

Key Responsibilities

  • Researched and evaluated different technologies to identify the most suitable approach for face-tracking.
  • Designed and implemented face-tracking functionality combining Python and Unity.
  • Adapted the Tella educational game to the latest Unity versions and ensured compatibility with the face-tracking system.
  • Prepared technical documentation and demos for stakeholders.

Tech Stack

Unity, C#, Git, Python, Blender.

Face Tracking Result

This screenshot shows the educational game we integrated with our face-tracking system. The pencil acts as the cursor and moves purely with head motion. To give the player clear aiming feedback, we also added the small chicken head overlay, which mirrors the user’s head movements in real time..

Meadipipe test implementation using Python

After an extensive investigation, we determined that the most suitable tool for obtaining facial landmarks was MediaPipe, a framework developed by Google that provides cross-platform, customizable machine learning solutions for real-time computer vision tasks, including face and hand tracking. Once we familiarized ourselves with its functionalities, we proceeded to explore existing MediaPipe integrations for Unity.
