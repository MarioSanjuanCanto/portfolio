Overview

A modern, responsive portfolio website built with HTML5, CSS3, and JavaScript. Features a clean design with smooth animations and mobile-first responsive layout. This project showcases my frontend development skills and understanding of modern web design principles.

Key Features

  • Responsive grid layout that adapts to all screen sizes
  • Smooth hover animations and transitions
  • Professional color scheme with modern aesthetics
  • Interactive navigation with smooth scrolling
  • Mobile-first responsive design approach
  • Optimized performance and accessibility

Tech Stack

  • Frontend: HTML5, CSS3, JavaScript
  • Design: Modern UI/UX principles
  • Responsive: Mobile-first approach with CSS Grid and Flexbox
  • Animations: CSS transitions and hover effects
  • Performance: Optimized images and efficient CSS
