Python Flask Web Application for MySQL

July 2025 – August 2025

Overview

I developed a complete web interface for managing MySQL databases that allows users to execute SQL queries and import data from CSV files in an intuitive way.

Key Features

  • Web interface for executing SQL queries in real-time
  • CSV import with drag & drop and delimiter configuration
  • User authentication system
  • Query execution history
  • SQL injection prevention with character escaping

Tech Stack

Python, Flask, MySQL, HTML5, CSS3, JavaScript(AI-generated interface), SQL Connector.

Project Link

https://gitlab.com/msancan3/mysqlpython

Images

MySQL Query Editor

Web interface for executing SQL queries in real-time, after you put the query in the web interface, the query is passed in the backed to the MySQL database and the results are displayed in the web interface.

Login to the web interface

In this case, when putting the login information, in the backend I use that information to create the connection to the MySQL database, trying to see if the user exists in the database. In case it exists, the user is logged in and the web interface is displayed, otherwise the user is not logged.

Import data from CSV

For importing the data, I read the information from the CSV file, and I use an INSERT INTO statement to insert the data into the MySQL database.
