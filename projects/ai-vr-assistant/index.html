AI Virtual Reality Assistant

2024 – 2025

Overview

After a lot of work, I'm excited to share that I've completed my project for the Interactive group: a virtual reality AI assistant designed to answer questions about the university.

How it works

  1. Put on the Virtual Reality headset to access the application
  2. Press the red button in front of you to activate the AI system
  3. Ask your question about the university in natural language
  4. An LLM model processes your query, retrieves information from a document, and then generates an audio response

Tools used

  1. FastAPI (Python) for text-to-speech and speech-to-text conversion
  2. Llama (LLM) to analyze and answer questions
  3. Google Docs accessed from Unity as the information source for the AI
  4. Unity as the development platform

Tech Stack

Python, Unity, FastAPI, Llama LLM, Google Docs API.
